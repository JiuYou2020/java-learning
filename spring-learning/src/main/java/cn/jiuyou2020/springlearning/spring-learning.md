# 关于学习spring和springboot的约定
> 我们知道springboot的特性之一便是约定大于配置，那么我们在学习spring和springboot的时候，也可以约定一些规则，以便更好的学习和使用spring和springboot。

1. 虽然本模块的项目名是spring-learning，但是实际上是关于spring和springboot的学习笔记，所以在这里约定一下，spring指的是spring framework，springboot指的是springboot框架。
2. 本模块的包名起名规则是参考《Spring Boot源码解读与原理分析》一书的目录的，内容也是参考这本书的。所以一共有三个大包
   - core_context: spring boot底层依赖的核心容器
   - life_cycle：spring boot的生命周期分析
   - dev_practices：spring与spring boot的开发实践
